# apps

Прикладные приложения платформы. Каждое приложение - **тонкая** оболочка: маршруты, конфигурация сборки, подключение доменов и модально-специфичного модуля визуализации.

| Каталог | npm name | Порт dev | Контур |
|---------|----------|----------|--------|
| `ultrasound/` | `ultrasound-app` | 3002 | 2D (УЗИ) |
| `cytology/` | `cytology-app` | 3001 | 2D (цитология) |
| `ct/` | `ct-app` | 3004 | DICOM (КТ) |
| `mri/` | `mri-app` | 3005 | DICOM (МРТ) |

Запуск: `npm run dev:<модальность>` из корня репозитория.

Модально-специфичный код размещается в `src/` приложения (`uzi_view`, `features/cytology-view` и т.д.) и не импортируется другими apps.
