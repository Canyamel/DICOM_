# configs

Общие конфигурации монорепозитория **medml-front**: пресеты TypeScript, Vite и соглашения ESLint.

| Каталог / файл | Назначение |
|----------------|------------|
| `tsconfig/` | Базовые `tsconfig` для приложений, библиотек и доменных пакетов |
| `vite/` | Общие настройки Vite для SPA-приложений (УЗИ, цитология) |
| `eslint/` | Описание единых правил линтинга (корневой `eslint.config.mjs`) |

Пакеты и приложения подключают пресеты через `extends`, чтобы не дублировать `compilerOptions` и алиасы `@medml/*`.
