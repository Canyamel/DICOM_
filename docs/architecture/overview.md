# Обзор архитектуры

Клиентская платформа **medml-front** - монорепозиторий семейства диагностических приложений (УЗИ, цитология, КТ, МРТ).

## Уровни

```
apps/      → прикладные приложения (маршруты, композиция)
domains/   → общие предметные контексты (пациенты, auth, layout)
lib/       → инфраструктура и модули визуализации
configs/   → общие пресеты TS / Vite / ESLint
tools/     → скрипты проверки границ и обслуживания workspaces
build/     → Docker, Nginx, поставка статики
```

## Контуры визуализации

- **2D** (`@medml/viewers`, Annotorious) - УЗИ, цитология
- **DICOM** (`@medml/ct-mri-viewer`, Cornerstone) - КТ, МРТ

Общий каркас (авторизация, списки, UI) переиспользуется; модально-специфичный код изолирован в приложениях или пакетах `lib`.
